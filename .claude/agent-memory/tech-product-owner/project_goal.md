Meu Negocio é projeto de portfólio open source de Ricardo, NÃO produto comercial em produção.

**Why:** Ricardo já o considera "quase finalizado" e a missão atual é fechar como peça apresentável (peça profissional para portfólio, demonstrar Clean Architecture, multi-tenancy, padrões Laravel), não evoluir features. O escopo NÃO PODE crescer ao infinito.

**How to apply:**
- Sempre considerar: "isso é demonstrável a um avaliador técnico em 5-10 minutos?".
- Cortar é decisão válida. Funcionalidade incompleta deve ser cortada OU completada mínima — nunca deixada parcial.
- Recusar features novas que não estejam no plano de fechamento (ver `docs/FECHAMENTO_PORTFOLIO.md`).
- Priorizar: README, .env.example, testes mínimos, CI básico, LICENSE, reset de senha — itens que avaliador técnico nota imediatamente.
- Fora de escopo confirmado: 2FA, gateway de pagamento, BI/relatórios, API REST mobile, observability, schema-per-tenant.
